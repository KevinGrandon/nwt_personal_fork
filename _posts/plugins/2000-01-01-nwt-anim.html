Animation

NWT ships with a simple and easy to use animation mechanism. Modern browsers receive CSS transitions.

Demo

Click Me!

Usage

{% highlight js %} // Get the element var el = n.one('.myel') // Do the animation el.anim({'margin-left':'600px', color:'#FF0000'}, 0.5) .wait(1) // After the first step, wait for one second .anim({'margin-left':'0px', color:'#000'}, 0.5) // Reset he animation after 1 second {% endhighlight %}

Reversing an applied animation

A common use case is to reverse an applied animation on click.

Click Me!

Usage

{% highlight js %} // Get the element var el = n.one('.myel') // Do the animation el.on('click', function(e) { // PopAnim will either run the last animation in reverse and return true, or return false this.popAnim() || this.anim({'margin-left':'600px', color:'#FF0000'}, 0.5) e.stop() }) {% endhighlight %}
